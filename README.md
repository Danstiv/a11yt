# A11YT

A Chrome extension that improves YouTube accessibility for screen reader users.

Coded by ai.

## Features

- Makes video links in the feed reachable by screen readers, so you can focus them and open videos in a new tab with Ctrl+Shift+Enter.
- Stops the player from announcing each new chapter title, so it no longer interrupts you while watching. *(off by default)*
- Gives the play/pause button a short, clear label that matches its current state. *(off by default)*

## Settings

Each feature can be turned on and off individually. Open the settings either by clicking the extension's toolbar button, or through its "Options" entry on chrome://extensions.

Only the feed links feature is enabled by default; the other two have to be turned on manually. Changes take effect after you reload open YouTube tabs.
